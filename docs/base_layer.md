# 图层

主页面右上角的图层工具提供了灵活切换显示图层的功能。
<img src="images/layer/layer1.png" alt="点击图层工具" width="80%" />

点击该工具的图标后，会出现一个复选框列表。
<img src="images/layer/layer2.png" alt="出现复选框列表" width="80%" />

里面包含了各种空间目标的类型，比如有效载荷、太空碎片等。
<img src="images/layer/layer3.png" alt="空间目标类型" width="80%" />

可以自行点击切换要显示图层。

<img src="images/layer/layer4.png" alt="切换图层" width="80%" />
