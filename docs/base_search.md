# 搜索工具

右上角的放大镜图标是一个搜索工具，可以按照名称或Norad ID检索数据库中的卫星。
<img src="images/search/search1.png" alt="点击搜索框" width="80%" />

点击打开搜索框后，输入想要检索的内容。
<img src="images/search/search2.png" alt="输入检索内容" width="80%" />

例如，我们输入“beidou”，会检索出所有名称中包含“beidou”的卫星，且不区分大小写。
<img src="images/search/search3.png" alt="输入北斗" width="80%" />
<img src="images/search/search4.png" alt="检索北斗卫星" width="80%" />

再输入"11436"，会检索出Norad ID为11436的卫星。
<img src="images/search/search5.png" alt="输入卫星编号" width="80%" />
<img src="images/search/search6.png" alt="查看卫星详情" width="80%" />
