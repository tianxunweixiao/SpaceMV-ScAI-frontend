# 系统总览

ScAI是一个用于对卫星星座进行统一管理，支持状态监控、传感器参数配置、覆盖性仿真分析、可视化展示的综合性平台。

系统的主要功能包括：
1）支持卫星/星座数据的定期更新和三维可视化展示；
2）支持自定义的星载传感器参数配置；
3）支持自定义的卫星/星座数据导入(示例文件-example.json)；
4）支持单星/星座级别的覆盖性仿真分析；
5）支持仿真结果的可视化展示和相关报告的输出；
6）支持相关AI工具的实时使用。

该系统的构建旨在开发一个融合大语言模型（LLM）及相关智能推理技术的综合性平台，致力于为业内提供智能化、交互式的星座管理工具，各模块的相关介绍和使用方法请查看用户手册。
