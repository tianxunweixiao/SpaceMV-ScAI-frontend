# 绘图模块

“绘图”模块主要包括“ECI”绘图和“ECF”绘图。选中某颗卫星后，点击“ECI”绘图后，会以三维的形式绘制该卫星在地心惯性坐标系下的轨迹。
<img src="images/plot/plot1.png" alt="点击ECI绘图" width="80%" />
<img src="images/plot/plot2.png" alt="ECI绘图界面" width="80%" />

点击“ECF”绘图后，会以三维的形式绘制该卫星在地心地固坐标系下的轨迹。
