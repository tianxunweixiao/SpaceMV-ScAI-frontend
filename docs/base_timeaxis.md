# 时间轴调整
页面左上角的显示时间可以自行调整，其中最前面的数字表示一年中的第多少天，后面的时间则是时分秒格式的UTC时间。

<img src="images/timeline/timeline1.png" alt="时间显示格式" width="80%" />

点击显示的时间后，可以自行修改调整。

<img src="images/timeline/timeline2.png" alt="时间调整界面" width="80%" />

<img src="images/timeline/timeline3.png" alt="时间修改示例" width="80%" />

<img src="images/timeline/timeline4.png" alt="时间设置完成" width="80%" />

修改好后，点击“完成”。

<img src="images/timeline/timeline5.png" alt="点击完成按钮" width="80%" />

原本显示的卫星的轨道会被自动外推到设置的时间点，并可视化显示在屏幕上。

<img src="images/timeline/timeline6.png" alt="更新后的卫星轨道" width="80%" />
