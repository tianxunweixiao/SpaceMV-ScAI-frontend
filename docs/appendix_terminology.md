# 相关术语

LLM - Large Language Model 的缩写，即大语言模型。它是一类基于深度学习的人工智能模型，能够理解、生成、翻译、推理和总结自然语言。

Agent - 在人工智能领域，特别是结合大语言模型（LLM）后，Agent通常指一种具备语言理解、任务规划与执行能力的自治系统，可以根据自然语言指令调用外部工具、接口或数据库完成复杂任务。

SSA - Space Situational Awareness，太空态势感知，指对地球轨道上的物体进行探测、跟踪、识别和预测其位置的能力，以及监测更广阔的太空环境以发现潜在风险或危险的活动。

SST - Space Surveillance and Tracking，空间监视与跟踪，SSA的核心组成部分，通过传感器对空间目标进行探测、跟踪和编目。

SEM - Space Environment Monitoring，空间环境监测，SSA的核心组成部分，监测空间天气和空间碎片环境的变化趋势。

SOIA - Space Object Identification and Analysis，空间目标识别与分析，SSA的核心组成部分，对空间目标进行识别、分类和分析。

LEO - Low Earth Orbit，低地球轨道，通常指距离地球表面 200 至 2000 公里的轨道。

TLE - Two-Line Element，双行轨道根数，一种用于描述地球轨道卫星位置和速度的简化数据格式。

POD - Precise Orbit Determination，精密轨道确定，通过高精度测量数据（如 GNSS、激光测距）解算得到的卫星轨道。

NPU - Neural-network Processing Unit，神经网络处理器，专门用于加速人工智能计算的处理器。

TOPS - Tera Operations Per Second，每秒万亿次操作，衡量 AI 芯片算力的单位。

TFLOPS - Tera Floating-point Operations Per Second，每秒万亿次浮点运算，衡量 AI 芯片浮点算力的单位。

INT8/FP16/FP32 - 整数 8 位/浮点 16 位/浮点 32 位，表示数据精度，用于模型量化以提高推理效率。

CANN - Compute Architecture for Neural Networks，华为昇腾异构计算架构。

ACL - Ascend Computing Language，昇腾计算语言，CANN 提供的底层编程接口。

OM - Offline Model，离线模型，昇腾 AI 处理器支持的模型格式。

ATC - Ascend Tensor Compiler，昇腾张量编译器，用于将模型转换为 OM 格式的工具。

GNSS - Global Navigation Satellite System，全球导航卫星系统。

NOAA - National Oceanic and Atmospheric Administration，美国国家海洋和大气管理局。

NASA - National Aeronautics and Space Administration，美国国家航空航天局。

ORDEM - Orbital Debris Engineering Model，轨道碎片工程模型。

ESA - European Space Agency，欧洲空间局。

MASTER - Meteoroid and Space Debris Terrestrial Environment Reference，流星体和空间碎片地球环境参考模型。

SGP4/SDP4 - Simplified General Perturbations models，简化通用摄动模型，用于快速计算卫星轨道。

NRLMSISE-00 - Naval Research Laboratory Mass Spectrometer and Incoherent Scatter Radar Exosphere model，美国海军研究实验室质谱仪和非相干散射雷达外逸层模型，一种大气密度模型。

JB2008 - Jacchia-Bowman 2008，一种大气密度模型

RESTful API - Representational State Transfer Application Programming Interface，一种设计网络应用程序的架构风格。

DOCX - Microsoft Word Open XML Document，微软 Word 文档格式。
